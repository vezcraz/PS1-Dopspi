      The easiest way is to use the data analysis toolpack

      Installing it: source: -https://support.office.com/en-ie/article/use-the-analysis-toolpak-to-perform-complex-data-analysis-6c67ccf0-f4a9-487c-8dec-bdb5a2cefab6

      1. Click the File tab, click Options, and then click the Add-Ins category.

        If you're using Excel 2007, click the Microsoft Office Button , and then click Excel Options

      2. In the Manage box, select Excel Add-ins and then click Go.

        If you're using Excel for Mac, in the file menu go to Tools > Excel Add-ins.

      3. In the Add-Ins box, check the Analysis ToolPak check box, and then click OK.

        • If Analysis ToolPak is not listed in the Add-Ins available box, click Browse to locate it.

        • If you are prompted that the Analysis ToolPak is not currently installed on your computer, click Yes to install it.

      Now tabulate the data like this:

      1. Click on the data analysis tools icon under the data tab and select regression> Ok

      2. Select the range for Y and X and the options that are applicable. Don't forget to select the line fit plots box

      3. click on OK and open the output sheet

      A scatter plot is thus generated along with the required estimates of intercept and the slope coefficients.
